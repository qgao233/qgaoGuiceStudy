# Bindings overview

通过创建一个`bind`，可以往`Guice map`中插入一条新的记录`entry`。

有这么多种bind：

* Linked Bindings
* Binding Annotations
* Instance Bindings
* @Provides Methods
* Provider Bindings
* Untargeted Bindings
* Constructor Bindings
* Built-in Bindings
* Just-In-Time Bindings
* Multibindings
* Restricting the Binding Source

这句话不懂：
The injector also includes bindings for the providers of its other bindings.
